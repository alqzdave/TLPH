# Changelog

All notable changes to the TLPH Inventory Management System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added
- Initial release of TLPH Inventory Management System
- Flask-based web application for inventory management
- Multiple service modules (seminar, fisheries, environment, forest, livestock, permits, wildlife, farm)
- Email notification support via Flask-Mail
- Firebase integration (currently disabled)
- Version information and package metadata
- Setup files for Python packaging (setup.py and pyproject.toml)

### Features
- Landing pages for various services
- API routes for data management
- Template-based UI with static assets
- Configuration management with environment variables
- SQLite database support
